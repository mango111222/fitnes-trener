# Фитнес-тренер Telegram бот

Автоматический бот на GitHub Actions для мотивации к тренировкам и контролю водного баланса.

## Возможности
- 📅 Ежедневная тренировка в 12:00 МСК
- 💧 Напоминание о воде в 18:00 МСК
- 📊 Цель на неделю: 5 тренировок

## Настройка
1. Создать бота через @BotFather
2. Добавить секреты в GitHub Actions:
   - TELEGRAM_BOT_TOKEN
   - TELEGRAM_CHAT_ID
